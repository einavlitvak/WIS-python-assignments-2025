BattleShip!

## Description

A  game of BattleShip!
The game generates 4 ships of random size (1-2) with random positions on a 5x5 grid.
The user's goal is to sink all ships by entering coordinates (X, Y).

## How to play

1. Clone the repository
2. Run the script: `python Battleship.py`
3. Enter X and Y coordinates (0-4) when prompted.
4. Keep playing until all ships are sunk!
